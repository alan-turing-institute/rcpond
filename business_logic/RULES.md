# Rules for triaging ServiceNow tickets

Has the user said "please" or "thank you", or similar polite language in their ticket?

- If yes, then add a work note, recommending that RCP should approve the request.
- If no, then add a work note, recommending that RCP should email the user to reword their request in more polite language before RCP can approve it.
