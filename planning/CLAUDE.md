## Repository Purpose

This is a planning repository for the RCPond project. It contains design documents and specifications — no source code, build system, or tests.

## Key Documents

- **SN.md** — Design specification for a Python module (`SN` class) that interfaces with ServiceNow. Defines the API for getting unassigned HPC/cloud access request tickets, retrieving full ticket details, posting work notes, and assigning tickets. Includes `Ticket` and `FullTicket` dataclass schemas.
